Common Questions

What is Terminal?

Terminal is a built-in Mac app that lets you run commands by typing them. Think of it as a text-based way to tell your computer what to do. You don't need to know how to use it beyond pasting the install command above.

It's asking for my password - is that safe?

Yes. Some tools (like Homebrew) need your Mac admin password to install system-wide software. Your password is never sent anywhere - it stays on your Mac. The cursor won't move while you type it - that's normal. Just type it and press Enter.

The install seems stuck - what do I do?

Some steps (like downloading OpenRV at 642 MB) can take several minutes. If nothing has happened for more than 10 minutes, press Ctrl + C to cancel, then try running the command again.

Can I uninstall it?

Yes - just delete the Comfy-Asset-Manager folder from your home directory. On Mac: open Finder, press Cmd Shift H to go to your home folder, and drag Comfy-Asset-Manager to the Trash.

What does the app actually install?

The installer adds these free/open-source tools if they're not already on your system: Homebrew (Mac package manager), Node.js (runs the app), FFmpeg (video processing), and npm packages (app dependencies). It optionally offers to install OpenRV (professional video player).

Do I need an internet connection?

Only for the initial install. After that, the app runs entirely on your local computer - no cloud, no accounts, no subscriptions.
